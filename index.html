Spielanleitung: Erstelle oder trete einem Spiel bei. Der Ersteller wählt den Spielmodus:
- Punkte-Modus: Wer die meisten Punkte hat, gewinnt
- Verlust-Modus: Wer keinen Block mehr platzieren kann, verliert
- Team-Modus: Alle Spieler arbeiten zusammen, Punkte werden addiert, Spiel endet wenn kein Block mehr platziert werden kann
Jeder Spieler platziert einen Block pro Runde.

Neue Funktion: Sieh eine Vorschau, wo dein Block platziert wird, bevor du ihn ablegst!
